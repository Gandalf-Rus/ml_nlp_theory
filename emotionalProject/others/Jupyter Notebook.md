
> [!INFO]  Для чего нужен Jupyter Notebook и связь с VS Code
>  В ML-проектах загрузка данных и обучение моделей могут занимать минуты или часы. 
>  Если писать всё в обычных `.py` скриптах, при малейшей правке в коде для построения графика тебе придется заново загружать весь датасет в память. 
>  Jupyter Notebook решает эту проблему: код разбивается на ячейки, которые можно выполнять независимо друг от друга . 
>  Ты один раз загружаешь данные в одной ячейке, а в следующих строишь графики, смотришь метрики и проверяешь гипотезы. 
>  Не нужно отказываться от VS Code — этот редактор имеет встроенную поддержку файлов `.ipynb` (формат Jupyter Notebook). Ты будешь писать интерактивный код прямо в привычном интерфейсе VS Code
