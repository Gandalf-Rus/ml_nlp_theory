
**Главное правило:** Когда мы вычисляем частную производную функции нескольких переменных по одной конкретной переменной, **все остальные независимые переменные воспринимаются как неизменные константы (обычные числа)**.

Из базового матанализа известно, что производная константы равна нулю ($c' = 0$).

**Пример на простой алгебре:**

Дана функция $F(x, y) = 5x + y^2$.

Нам нужно найти частную производную по $x$ (обозначается $\frac{\partial F}{\partial x}$).

- Мы смотрим на слагаемое $5x$. Его производная по $x$ равна $5$.
    
- Мы смотрим на слагаемое $y^2$. В нем нет переменной $x$. Для $x$ это просто статичное число (как если бы там стояло $42$). Производная числа равна $0$.
    
- Итог: $\frac{\partial F}{\partial x} = 5 + 0 = 5$.
    

**Перенос на формулу LSTM:**

Наше уравнение долгосрочной памяти:

$$C_t = (f_t \odot C_{t-1}) + (i_t \odot \tilde{C}_t)$$

Мы ищем частную производную строго по переменной $C_{t-1}$:

1. Первое слагаемое содержит $C_{t-1}$. При дифференцировании переменная "исчезает", оставляя только коэффициент перед ней: $f_t$.
    
2. Второе слагаемое $(i_t \odot \tilde{C}_t)$ в своем явном виде вообще не содержит $C_{t-1}$. Для текущей операции дифференцирования весь этот блок — просто гигантская константа.
    
3. Производная этой константы обнуляется.
    

_(Академическое уточнение: технически $i_t$ и $\tilde{C}_t$ имеют скрытую косвенную зависимость от $C_{t-1}$ через вектор $h_{t-1}$, но при расчете основного «шоссе градиентов» берется именно **явная частная производная** $\frac{\partial C_t}{\partial C_{t-1}}$, которая фиксирует остальные выходы вентилей как константы. Именно этот математический путь позволяет градиенту течь сквозь время без матричных перемножений)._
